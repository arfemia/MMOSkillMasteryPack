Full translations in all 9 languages, and the Mastery Infusion shop offer actually works now.

- **New: every string translated.** Track titles, node names and descriptions, quest titles and flavor text, achievement names, currency names, and the shop offer now ship in English, German, Spanish, French, Hungarian, Italian, Brazilian Portuguese, Russian, and Turkish. Node tier names ("Sharpened", "Vampiric", "Eternal", ...) are translated once per tier and composed with the localized track title by the plugin (1.3.0+), so even custom-skill tracks you fan out yourself stay translated. Anything a translation misses falls back to English per key.
- **Fixed: the Mastery Infusion offer never appeared in the shop.** 1.0.1 shipped it in the wrong content folder with a stale Control key, so the offer was dead content. It now shows up as intended when the Bounty Pack is installed on plugin 1.3.0+.
- **Reward names no longer pin to English.** Reward line-items drop their baked-in English names; the plugin auto-renders each line ("+5 Mastery Points", "+2,000 Mining XP") in the player's language.
